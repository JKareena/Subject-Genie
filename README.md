# Subject Genie

Subject Genie is an AI-based elective recommendation system designed to help students choose the most suitable subjects based on their interests and academic preferences.

## Features

* Personalized subject recommendations
* User-friendly interface
* Responsive frontend design
* AI-assisted decision support

## Tech Stack

* Python
* Flask
* HTML, CSS, JavaScript
* Figma (for UI/UX design)

## Description

The project focuses on improving student decision-making by combining logical recommendation systems with an intuitive user interface. It demonstrates both frontend development and basic backend integration.
